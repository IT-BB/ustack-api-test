# S3

## 获取S3登录信息 - S3Login

### 请求参数

| 字段名 | 类型 | 描述信息 | 示例 | 参数来源 | 必填 |
|:---|:---|:---|:---|:---|:---|
| **Region** | string | 地域 | [] | [] | **Yes** |
| **CompanyID** | integer | 租户ID | [] | [] | No |
| **Bucket** | string | 桶名称 | [] | [] | No |
| **ID** | string | 目标S3的ID | [] | [] | No |
| **Scenes** | string | 使用场景 | [] | [] | **Yes** |

### 响应字段

| 字段名 | 类型 | 描述信息 | 示例 |
|:---|:---|:---|:---|
| **AccessKey** | string | 密钥ID | [] |
| **SecretKey** | string | 密钥 | [] |
| **Endpoint** | string | 访问地址 | [] |
| **SessionID** | string | S3会话认证信息 | [] |
| **Bucket** | string | 桶名称 | [] |
